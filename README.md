# easy1400是基于GAT 1400-2017公安视频图像信息应用系统标准协议实现的快速对接平台
因公司的对接平台实在太过拉垮，甚至连对接页面也没有，纯靠postman对接实属无语。公司也没计划开发一个像样的平台，所以决定利用自己闲暇时间开发一个稍微像点样的对接平台！
